# 🚀 DevFolio — Terminal Portfolio

DevFolio is a retro CRT-style portfolio website that simulates a real terminal experience.
Visitors can run commands to explore your bio, skills, projects, experience, and contact links.

---

## 🌍 Live Demo

https://sevanton88.github.io/devfolio/

---

## ✨ Features

- Retro CRT visual design with scanlines, glow, and terminal chrome
- Interactive command-line interface with command history support
- Boot sequence and typing animations for immersive UX
- Portfolio data rendered through commands (about, skills, projects, experience, contact)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)

---

## ⚡ Getting Started

1. Clone the repository  
2. Open `index.html` in your browser  

---

## 🎯 Project Goal

Practice advanced DOM manipulation, UI state handling, and animation timing in vanilla JavaScript while creating a unique personal portfolio experience.
